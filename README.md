NovaAI
NovaAI is a modern AI landing page built using HTML, CSS, and JavaScript.
I created this project to practice frontend development, responsive design, and building clean modern UI layouts.
The website has a premium dark theme with glowing effects, smooth layouts, pricing cards, testimonials, and an AI assistant section.
